VOICEGRAM

Your voice, translated into visual form.

VOICEGRAM은 마이크나 오디오 파일의 소리를 실시간으로 분석해, 목소리의 특징을 그래픽 형태와 패턴으로 변환하는 인터랙티브 비주얼 도구입니다.

Mosaic Mode

목소리의 리듬과 밀도를 그리드 기반의 색면 패턴으로 변환합니다. 여러 도형을 만드는 모드가 아니라, 소리를 정제된 패턴 포스터처럼 번역하는 모드입니다.

Shape Mode

목소리의 높낮이, 밝기, 에너지, 변화량을 분석해 다양한 그래픽 모티프를 생성합니다. 낮고 굵은 소리는 둥근 형태로, 높고 밝은 소리는 뾰족한 형태로, 변화가 큰 소리는 리듬감 있는 점과 선 요소로 표현됩니다.

How to use

  • MIC: 마이크 입력으로 실시간 그래픽을 생성합니다.
  • FILE: 오디오 파일을 업로드해 그래픽을 생성합니다.
  • Shape / Mosaic: 서로 다른 시각화 방식을 선택합니다.
  • Background: Shape 모드의 배경을 white 또는 black으로 선택합니다.
  • Canvas: 캡처할 화면 비율을 선택합니다.
  • Capture / Record: 정지 이미지나 영상을 저장합니다.
