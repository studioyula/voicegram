# 라인 스윕(Line Sweep) 지도 가이드

> Cursor AI + AE MCP 환경에서 라인 스윕 모션을 가르치는 순서와 핵심 키워드 정리

---

## 라인 스윕이란?

곡선 패스를 따라 선이 연장되듯이 그려지는 모션 기법.
마치 누군가 펜으로 곡선을 쭉 그리는 것처럼 보이는 것이 핵심.

---

## 반드시 언급해야 할 키워드

| 키워드 | 설명 |
|--------|------|
| **Trim Paths** | 셰이프 레이어에 추가하는 속성. End 값으로 선의 길이를 제어 |
| **End 0% → 100%** | 선이 처음부터 끝까지 연장되는 핵심 키프레임 |
| **곡선 패스 (Bezier Curve)** | 직선이 아니라 베지어 탄젠트가 있는 부드러운 커브 |
| **원본 패스 유지** | 패스 모양 자체는 절대 변형하지 않음. 패스에 키프레임 금지 |
| **Open Path** | closed가 아닌 열린 패스여야 연장선 느낌이 남 |
| **Round Cap (둥근 끝)** | 스트로크 끝을 둥글게 해서 펜 드로잉 느낌을 살림 |
| **이징 (Easing)** | 시작과 끝에서 속도 변화를 줘서 자연스러운 드로잉감 |
| **스태거 (Stagger)** | 여러 선을 순차적으로 그릴 때 시작 시점을 엇갈리게 배치 |

---

## 지도 순서

### Step 1 — 개념 설명

> "라인 스윕은 곡선 패스를 따라 선이 연장되듯이 그려지는 모션이야. 마치 펜으로 곡선을 쭉 긋는 것처럼 보이는 거지."

- 직선이 아니라 **곡선**임을 강조
- 레이어가 이동하거나 스케일되는 게 아님을 명확히
- Trim Paths라는 속성이 핵심 도구임을 알려줌

### Step 2 — 데모 컴프 분석

> "AE에 LineSweep_curve_demo라는 컴프가 있어. 이 컴프를 자세히 분석해봐."

커서가 직접 스캔해서 구조를 파악하게 함:
- 셰이프 레이어 안에 곡선 패스가 있음
- Trim Paths > End가 0→100 키프레임
- 패스 자체에는 키프레임 없음 (원본 유지)
- 스트로크: 흰색, Round Cap

### Step 3 — 똑같이 만들어보기

> "분석한 걸 기반으로 네가 직접 새 컴프를 만들어서 비슷한 걸 만들어봐."

조건:
- 곡선 패스를 자유롭게 디자인 (단, 반드시 곡선)
- Trim Paths로 연장선 애니메이션
- 흰색 스트로크, 둥근 끝
- 이징 부드럽게

### Step 4 — 검증

만든 결과를 ae_scan / ae_execute로 점검:
- 패스에 키프레임이 없는지 (원본 변형 금지)
- Trim End 0→100 키프레임이 있는지
- 베지어 탄젠트가 있는 곡선인지 (직선 아닌지)
- Round Cap인지

### Step 5 — 다중 라인 응용

> "이번엔 곡선 3개를 만들어서 순차적으로 그려지게 해봐."

- 여러 곡선이 차례차례 그려지는 스태거 패턴
- 첫 번째 선이 다 그려지면 → 두 번째 시작 → 세 번째 시작
- 각 선마다 독립적인 Trim Paths

### Step 6 — 자유 응용

> "지금까지 배운 걸 응용해서 네가 자유롭게 만들어봐."

- 곡선 방향, 개수, 구도를 자유롭게
- 단, 반드시 곡선 + Trim Paths + 원본 패스 유지

---

## 커서에게 시킬 때 주의사항

1. **"직선 금지"를 명확히** — 안 그러면 직선 패스로 만듦
2. **"패스 모양 자체는 변형하면 안 돼"** — 패스에 키프레임 넣는 실수 방지
3. **"Position/Scale/Opacity 애니메이션은 필요 없어"** — Trim Paths만 사용
4. **수치를 직접 주지 않기** — 디자인과 학생이 자연어로 말하듯이 지시
5. **만든 후 반드시 검증** — ae_scan으로 결과 확인시키기

---

## 잘못 이해하기 쉬운 패턴

| 잘못된 이해 | 올바른 이해 |
|-------------|-------------|
| 레이어들이 순차 등장 (Scale 0→100) | 선이 패스를 따라 연장됨 (Trim End 0→100) |
| 레이어가 슬라이드인 (Position 이동) | 패스 위치는 고정, Trim만 변화 |
| 직선으로 스윕 | 곡선(베지어 커브)으로 스윕 |
| 패스 모양이 변하면서 그려짐 | 패스 원본 유지, Trim으로만 노출 범위 조절 |
